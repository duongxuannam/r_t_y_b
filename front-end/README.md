# Todo Pulse Front-End

React + Vite front-end for the Rust Todo API.

## UI stack

- Tailwind CSS
- shadcn/ui components (replaces DaisyUI)

## Getting started

```bash
bun install
bun run dev
```

## Monorepo

From the repo root, you can run both apps together:

```bash
bun install
bun run dev
```

## Notes

- Theme switching uses the `data-theme` attribute on `<html>`.
